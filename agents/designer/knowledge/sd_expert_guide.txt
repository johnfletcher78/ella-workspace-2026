# Stable Diffusion - Designer Agent Expert Guide

## Core Concepts

### What is Stable Diffusion?
- Open-source AI image generation model
- Latent diffusion: Works in compressed latent space (faster)
- Version 1.5, 2.1, XL, SDXL Turbo
- Local execution: Privacy, no API costs

## Model Types

### Base Models
- **SD 1.5**: Versatile, most compatible
- **SD 2.1**: Better anatomy, less flexible
- **SDXL**: 1024x1024 native, higher quality
- **SDXL Turbo**: Real-time generation (1-4 steps)

### Specialized Models (LoRA)
- **Photorealistic**: Realistic Vision, EpicRealism
- **Anime**: Anything V5, Counterfeit
- **Artistic**: DreamShaper, Deliberate
- **Specific styles**: Character LoRAs, style LoRAs

## Prompt Engineering

### Structure
```
[Subject] + [Action] + [Environment] + [Style] + [Quality tags]
```

### Example
```
Cyberpunk dashboard interface, holographic displays, 
neon blue and purple lighting, futuristic UI design,
high detail, 8k, sharp focus, professional photography
```

### Positive Prompt Tips
- Be specific: "red sports car" vs "car"
- Use quality tags: "8k, highly detailed, masterpiece"
- Include style: "digital art, concept art, photorealistic"
- Lighting: "soft lighting, dramatic shadows, volumetric"

### Negative Prompts (What to avoid)
```
blurry, low quality, distorted, deformed, 
bad anatomy, watermark, signature, cropped
```

## Parameters

### Sampling Steps
- 20-30: Standard quality
- 30-50: Higher quality (slower)
- 4-8: Turbo models only

### CFG Scale (Classifier Free Guidance)
- 7-9: Balanced (recommended)
- 10-15: More literal to prompt
- 3-5: More creative/freestyle

### Resolution
- SD 1.5: 512x512 (native), 768x768 (upscale)
- SDXL: 1024x1024 (native)
- Aspect ratios: 16:9, 9:16, 4:3, etc.

### Samplers
- **DPM++ 2M Karras**: Fast, good quality
- **Euler a**: Creative, varied results
- **DPM++ SDE Karras**: Best quality (slower)

## Local Setup (RTX 4080)

### Installation
```bash
# Clone WebUI
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui
cd stable-diffusion-webui

# Install models to models/Stable-diffusion/
# Download from HuggingFace or CivitAI

# Launch with API
python launch.py --api --listen --port 7860
```

### API Usage
```python
import requests

response = requests.post('http://localhost:7860/sdapi/v1/txt2img', json={
    'prompt': 'modern dashboard UI, dark theme',
    'negative_prompt': 'blurry, low quality',
    'steps': 30,
    'cfg_scale': 7,
    'width': 1024,
    'height': 1024,
    'sampler_name': 'DPM++ 2M Karras'
})

image_data = response.json()['images'][0]
```

## Pro Tips

1. **Seed Control**: Use same seed for consistency
2. **Prompt Variations**: Small changes = big results
3. **Batch Generation**: Generate 4-8, pick best
4. **Post-Processing**: Use Photoshop/Figma for final polish
5. **Upscaling**: Generate small, upscale 2x or 4x

## Ethics & Copyright
